# UnitConverterApp

A modern, intuitive Android application built to perform quick and accurate unit conversions between various metric and imperial measurements.

## 🚀 Features
* **Dynamic Conversion:** Supports real-time conversion for Centimetres, Meters, Feet, and Millimeters.
* **State Management:** Leverages `remember` and `mutableStateOf` for efficient UI updates and state persistence.
* **Modern UI:** Designed with **Jetpack Compose** and **Material 3** for a sleek, responsive user experience.
* **Error Handling:** Includes robust input validation using the Elvis operator (`?:`) to handle null or invalid numerical entries gracefully.

## 🛠 Tech Stack
* **Language:** [Kotlin](https://kotlinlang.org/)
* **UI Framework:** Jetpack Compose (Material 3)
* **Build System:** Gradle (Kotlin DSL)
* **Architecture:** Component-based UI with reactive state updates.

## 📖 Key Learnings
This project helped me solidify my understanding of the **Jetpack Compose lifecycle** and the use of **Dropdown Menus** and **Outlined Text Fields**. I also practiced implementing custom logic for mathematical conversions within a mobile environment.

## 📥 Installation
1. Clone the repository: `git clone https://github.com/bhavyadesai12/UnitConverterApp.git`
2. Open in **Android Studio**.
3. Run on a device or emulator (Minimum SDK: 24).
